  常见问题
═══════════════════════════════════════════════════════════════

Q: 按钮仍然不可见？
A: 
  1. 强制刷新：Ctrl+Shift+R
  2. 清除缓存：Ctrl+Shift+Delete
  3. 使用无痕模式：Ctrl+Shift+N
  4. 重启开发服务器：
     Ctrl+C (停止)
     npm run dev (重启)

Q: 科技感皮肤没有变化？
A:
  1. 确保强制刷新（Ctrl+Shift+R）
  2. 检查 dist 目录是否已更新
  3. 检查 CSS 文件是否正确加载

Q: 如何重新构建？
A:
  npm run build
